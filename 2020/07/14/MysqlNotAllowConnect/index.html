mysql：不允许被远程访问解决方法

Host ‘XXXXXXXX’ is not allowed to connect to this MySQL server
不允许被远程访问

第一步输入：use mysql;

第二步输入：select host from user where user=’root’;
输入以上两个命令查看主机配置信息

第三步输入：update user set host = ‘%’ where user =’root’;
将host设置为通配符模式% Host设置“%”便可远程访问了

第四步输入：flush privileges;
是更改的配置立即生效

然后就可以正常访问了
